常见的数值函数如下

| **函数**     | 功能                  |
| ---------- | ------------------- |
| ceil（x）    | 向上取整                |
| floor(x)   | 向下取整                |
| mod(x,y)   | 返回x/y的模             |
| rand()     | 返回0~1内的随机数          |
| round(x,y) | 求参数的x的四舍五入的值，保留y位小数 |
练习：通过数据库的函数，生成一个六位数的随机验证码。
select  lpad(round(rand()*1000000 , 0), 6, '0');
